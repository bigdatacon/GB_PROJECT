Рекомендательная система товаров пользователям (без холодного старта - все пользователи ранее покупали товары)

1. Условия для прогноза:
# Целевая метрика - money precision@5. Порог для уcпешной сдачи проекта money precision@5 > 20%
# Бизнес ограничения в топ-5 товарах:

# Для каждого юзера 5 рекомендаций (иногда модели могут возвращать < 5)

# 2 новых товара (юзер никогда не покупал)

# 1 дорогой товар, > 7 долларов

# Все товары из разных категорий (категория - department)

# Стоимость каждого рекомендованного товара > 1 доллара

2. Описание расположения файлов: 
Итоговая модель файл MODEL
предобработка датасетов в папке src в файлах utils_i и recomenders_i, в файле metrics расчет целевых метрик

3. Описание построения модели: 
двухуровневая модель, за основу выбора товаров для рекомендации взят метод 1 из 4 методов библиотеки implisit,  дающий больший скор по recall_K(методы ниже):
get_own_recomendation(товары ранее приобретенные пользователем)
get_als_recommendations (расчет релевантности товара методов ALS) 
get_similar_items_recommendation (метод поиска похожего товара)
get_similar_users_recommendation (товары купленные похожим и пользователями) 

далее моделью второго уровня ( LGBMClassifier) выбирается топ 5 товаров для рекомендации

4. Финальные цифры по модели: 
precision_5 = 11.8%
money_precision_5 = 43%
